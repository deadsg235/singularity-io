# Blockchain (Rust Solana Integration)

This directory contains the Rust-based custom blockchain logic and Solana integration for Singularity.io.

## Getting Started

1.  **Navigate to the blockchain directory:**
    ```bash
    cd blockchain/solana_integration
    ```
2.  **Build the project:**
    ```bash
    cargo build
    ```
3.  **Run tests (if any):**
    ```bash
    cargo test
    ```

## Structure

*   `solana_integration/`: The main Rust project for Solana integration.
